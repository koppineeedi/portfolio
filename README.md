## 🛠️ Tech Stack & Features

### Website Design:
* **Frontend:** Responsive layout built with vanilla **HTML5** and **CSS3**
* **Interactivity:** **JavaScript** for navigation controls, theme toggling, and interactive overlays
* **Icons:** **Lucide Icons CDN** for clean and modern cybersecurity/shield visual assets
* **Micro-interactions:** Hover effects, smooth page transitions, and animations

### Features:
1. **Interactive Project Modals:** Clicking on any project card triggers a popup modal detailing the project objectives, methodology, and outcome.
2. **Dynamic Navigation Menu:** Highlighting active sections as you scroll and full mobile support (hamburger menu).
3. **Contact Form Integrations:** A validated contact interface that triggers a confirmation toast notification.

## 🚀 How to Run Locally

If you want to view or test this portfolio on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/koppineeedi/portfolio.git
   ```
2. **Navigate into the folder:**
   ```bash
   cd portfolio
   ```
3. **Open in browser:**
   Double-click the `index.html` file or run it via VS Code's **Live Server** extension.
